Grammar File Structure

Grammar Files ››

The grammar file is broken down into three major sections: the header, the parameters, and the rules.


The Header

The header has basic information such as the name, description and author of the grammar. Additionally, it contains the parameters (if any exist for the grammar).


Properties

Element

Description

Values

Name

The name of the grammar

Description

A short description of what the grammar produces

Category

The category that the grammar fits under        Commonly NPC, Name, or Other

Author

The name of the author

SupportsMaxLength

Does this grammar support max length?

true/false


Sample


<Grammar xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"

        xmlns:xsd="http://www.w3.org/2001/XMLSchema"

        xsi:type="AssignmentGrammar" >

   <name>US Names</name>

   <category>Name</category>

   <author>Lance Boudreaux</author>

   <description>A full name generator that presents names based on the top 300 male first, female first, and surnames.</description>

   <supportsMaxLength>false</supportsMaxLength>


The Parameters

There is some limited functionality allowing for parameters. In the future this functionality will be increased.

Properties

Element/Attribute

Description

Values

Parameter

A single parameter

[name]

The name of the label the parameter value will replace

[display]

The name to display for the parameter

[type]

The type of parameter

List/Text/Checkbox

option

The options available if the parameter is of list type

               

Sample


<parameters>

   <parameter name="Gender" display="Gender" type="List">

       <option>Any</option>

       <option>Male</option>

       <option>Female</option>

   </parameter>

</parameters
