Calculation Functions

Grammar Files ››

Below you can find a list of the custom calculation functions available for all grammar types.  For the standard functionality available please visit the NCalc website at https://ncalc.codeplex.com/


Function

Description

Roll(Number, Die, [options])

Rolls the type of die specified the number of times specified and returns the result.  For more information on the options, please see the section on Die Roll Options below.

TargetRoll(Number, Die, Target, [options])

Rolls the type of die specified and compares each roll against the target returning the number of successes or failures.  For more information about the options, please see the section on Die Roll Options below.

Rnd(Max)

Returns a random integer between 1 and Max inclusively.

Rnd(Min,Max)

Returns a random integer between Min and Max inclusively.

ToText(Number)

Returns the text representation of the Number (ex: ToText(7) => seven)

ToOrdinal(Number)

Returns the ordinal version of the Number (ex: ToOrdinal(11) => 11th)

UCase(String)

Returns the string in all caps. (ex: UCase("orcs") => ORCS)

LCase(String)

Returns the string in all lower case. (ex: LCase("ORCS") => orcs)

TCase(String)

Returns the string with all words capitalized. (ex: PCase("john henry doe") => John Henry Doe)

Format(Value, String)

Returns the Value formatted using the String format.  See Microsoft's documentation on the Format method for more details. (ex: Format(10000,"#,##0") => 10,000)

Generate(GrammarFileName or Path, [MaxLength])

Executes the grammar with the file name of GrammarFileName.rnd.xml providing the optional maxLength.  Does not currently support generators with parameters.




Die Roll Options:

'DL' : Drop the lowest die roll.  This can be followed by a number to indicate the number of dice to drop.


Examples:

Roll(2,20,'DL') will roll 2 20-sided dice and drop the lowest roll as is done with Advantage in D&D 5e

Roll(5,6,'DL',2) will roll 5 6-sided dice and drop the lowest 2 rolls as is one option for rolling attributes in D&D


Sample Results of the Examples:

Die Results: 15, 10 Result: 15

Die Results: 3, 4, 1, 6, 5 Result: 15


'DH' : Drop the highest die roll.  This can be followed by a number to indicate the number of dice to drop.


Examples:

Roll(2,20,'DH') will roll 2 20-sided dice and drop the highest roll as is done with Disadvantage in D&D 5e

Roll(5,10,'DH',4) will roll 5 10-sided dice and drop the highest 4 leaving only the lowest roll


Sample Results of the Examples:

Die Results: 15, 10 Result: 10

Die Results: 2, 5, 1, 9, 10 Result: 1


'EX' : Exploding dice re-rolls all maximum rolls and adds the results


Example:

Roll(4,10,'EX') will roll 4 10-sided dice and re-roll all 10's


Sample Results of the Example:

Die Results: 4, 10, 3, 6 + 10 Result: 33


'CX' : Compound exploding dice re-rolls all maximum rolls until none remain and adds the results


Example:

Roll(4,10,'CX') will roll 4 10-sided dice and re-roll all 10's


Sample Results of the Example:

Die Results: 8, 10, 2, 10 + 10, 8 + 10 + 3 Result: 61


'RB' : Re-roll below a given result


Example:

Roll(4,10,'RB',5) will roll 4 10-sided dice and re-roll all results under 5


Sample Results of the Example:

Die Results: 3, 10, 5, 2 + 6, 8 Result: 29


'GT' : Compare results to the target number provided looking for rolls over the target and return success or failure


Example:

Roll(3,10,'GT',6) will roll 3 10-sided dice and return success or failure depending on the number of rolls over 6 as is done in New World of Darkness games


Sample Results of the Example:

Die Results: 1, 1, 6, 9, 10 Result: Success (3)


'LT' : Compare results to the target number provided looking for rolls under the target and return success or failure


Example:

Roll(2,20,'LT',10) will roll 2 20-sided dice and return success or failure depending on the number of rolls under 10.


Sample Results of the Example:

Die Results: 10, 15 Result: Failure (-2)


'R1' : For target rolls ('GT','LT') this enacts the rule of one where one's remove successes from the result


Example:

Roll(4,10,'GT',6,'R1') will roll 4 10-sided dice and remove successes equal to the number of one's rolled and return success or failure as is done in Old World of Darkness


Sample Results of the Example:

Die Results: 1, 1, 10, 4 Result: Failure (-1)
