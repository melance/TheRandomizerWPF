Introduction ››

The Randomizer is a tool designed with table top role players in mind though it has use well beyond its intended audiences. It is capable of generating lists of names, dungeon dwellings, cities, characters, and so much more. Now, there are many sites and tools for generating these various things so why create a new one? The greatest strength of The Randomizer is that it can be customized to create almost any random thing that you can imagine and then share them with others as you would like. The options are very near limitless.
