Html Help Compiler Error Message Reference

EXITCODE #: Compiler unexpected termination.

Problem: Error is described directly in message description.

Description: Usually this message is displayed when compiler (hhc.exe) could not found the project file to compile. But in fact, this can be any other low level compiler error, that occures during compilaiton.

Solution: First check path (and possibly command line parameters) to the Html Help Compiler (defined in application options dialog) or ask your system administrator what to do.
