Html Help Compiler Error Message Reference

HHC3013: The file "filename" already exists. Please change the name of the split file to something else. The existing file will be stored until this problem is resolved. No additional component files will be stored.

Problem: The split file command specifies an HTML file name that already exists.

Result: The HTML file will not be split into new HTML files.

Solution: Rename the split file to give it a unique name.
