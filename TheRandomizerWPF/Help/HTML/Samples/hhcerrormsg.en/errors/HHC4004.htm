Html Help Compiler Error Message Reference

HHC4004: The option "entry" is not recognized.

Problem: An entry is listed in the [OPTIONS] section of the project file that the compiler does not recognize.

Result: The option will be ignored.

Solution: This is normally caused by using an older version of the compiler with a project file created by a newer version of HTML Help Workshop or other authoring tool. If this is the case, update to the most recent compiler, otherwise, delete the line containing the invalid option.
