Html Help Compiler Error Message Reference

HHC5001: An error occurred while reading "filename".

Problem: The compiler was unable to completely read the file.

Result: The .chm file is corrupt and should not be used.

Solution: This error is usually caused either by a corrupt hard disk or the inability to read a file from the project on a network drive because the network is down. If the file is on your hard disk, run ScanDisk to check your hard drive. If the file is on a network, check to see that the network is up, and if it is, contact your network administrator to find out why the file cannot be read.
