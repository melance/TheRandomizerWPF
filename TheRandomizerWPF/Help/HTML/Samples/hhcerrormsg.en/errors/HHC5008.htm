HHC5008: An unknown error occured while creating the index.

Problem: A problem occurred while creating a binary index.

Result: The .chm file is corrupt and cannot be used.

Solution: If you don't have a specific reason for using a binary index, you can turn it off to see if the .chm file compiles correctly. You should also check to be certain you have sufficient disk space on the drive where the .chm file is being created.
