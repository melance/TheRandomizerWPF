HHC4005: The following window line does not contain an '=' character separating the option and its value: "window entry."

Problem: A line in the [WINDOWS] section of the project file does not contain a '=' character.

Result: The window definition will be ignored.

Solution: Add a '=' character between the name of the window type and its definition. If you intended this line to be a comment, then you need to add a semi-colon to the beginning of the line.
