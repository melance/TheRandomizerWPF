Html Help Compiler Error Message Reference

HHC4009: Cannot open the file "filename" in the [Text Popup] section.

Problem: A file was specified in the [TEXT POPUPS] section that does not actually exist.

Result: The file will not be compiled into the .chm file, and any program trying to access the file in the .chm file will fail.

Solution: Change the name or path of the file to point to the correct file.
