Html Help Compiler Error Message Reference

HHC3015: An alias has been created to "filename" but the file does not exist

Problem: In the [ALIAS] section of the project file, you have created an alias to a file that does not exist. Alternatively, the file exists butyou have mapped to an anchor in the topic (e.g., filename.htm#bookmark).

Result: The alias is broken—any program specifying the mapped id that was aliased to this file will fail.

Solution: Change the name of the HTML file to the correct path or name, or remove the alias and mapped id. If the error is caused by a bookmark, you can ignore the error; the link should open the page at the bookmark.
