Html Help Compiler Error Message Reference

HHC4007: The section "section name" is unrecognized.

Problem: A section within the project (.hhp) file has an unrecognized name. Sections are lines beginning with a left bracket.

Result: The entire section will be ignored.

Solution: If the section name is incorrectly typed, change it to the correct section name. If the project file was created by an authoring tool, then you must use that authoring tool to compile the file (in order to process that section).
