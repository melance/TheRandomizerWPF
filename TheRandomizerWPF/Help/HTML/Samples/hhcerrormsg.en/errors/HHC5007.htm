Html Help Compiler Error Message Reference

HHC5007: Fatal Navigational Compilation Error. This is likely the result of an invalid .hhc file.

Problem: The compiler encountered a problem while creating a binary TOC.

Result: The .chm file is corrupt and cannot be used.

Solution: We don't know what causes this error and were unable to create an HHC file that would cause it. If you encounter this error, we'd love to get a copy of the .hhc file that produced the error. Please email it to kthelp@workwrite.com. Thanks!
