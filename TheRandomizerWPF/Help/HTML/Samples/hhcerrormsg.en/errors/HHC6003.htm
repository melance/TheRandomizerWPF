Html Help Compiler Error Message Reference

HHC6003: The file itircl.dll has not been registered correctly.

Problem: The compiler was unable to load either itircl.dll or itcc.dll.

Result: Full-text search, KLinks, and ALinks all require this file. If this error message occurs, all of these functions are broken in the HTML Help system you just compiled.

Solution: The problem is usually that itircl.dll or itcc.dll are not registered correctly (itircl.dll requires itcc.dll file when compiling). To register the file from KeyTools 2.0, choose Find Files from the Tools menu, and then choose HTML Help components. After the list of components builds, select the version of itircl.dll that you want to register, and then click the Register button. If the problem persists, register itcc.dll.

If you do not have KeyTools 2.0 or later, you will need to reinstall HTML Help Workshop.
