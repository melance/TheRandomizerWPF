Html Help Compiler Error Message Reference

HHC5009: Infotype compilation has failed. A maximum of 65,536 unique infotype combinations has been exceeded.

Problem: You have specified more then 65,536 information types.

Result: The .chm file is corrupt and cannot be used.

Solution: Reduce the number of information types you are using to a maximum of 65,536. And if you get this message, are you nuts?! :-)
