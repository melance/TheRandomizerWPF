Html Help Compiler Error Message Reference

WINERR #: System error message.

Problem: This is not a Html Help Compiler message and the problem is described directly in message description.

Description: Usually this message is displayed when compiler (hhc.exe) is not found, or when user access rights are insuffucient to execute the compiler. But in fact, this can be any system error, that occures during compilaiton.

Solution: First check path to the Html Help Compiler (defined in application options dialog) or ask your system administrator what to do.
