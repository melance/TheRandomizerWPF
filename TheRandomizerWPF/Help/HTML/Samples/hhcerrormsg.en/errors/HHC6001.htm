Html Help Compiler Error Message Reference

HHC6001: An unknown error occured while creating the full-text search data.

Problem: The compiler was unable to create the necessary data to enable full-text search.

Result: The .chm file is corrupt and cannot be used.

Solution: Since the cause is unknown, resolving the problem will require some trial and error. First, make certain there is sufficient disk space on the drive where your .chm file is being created.
