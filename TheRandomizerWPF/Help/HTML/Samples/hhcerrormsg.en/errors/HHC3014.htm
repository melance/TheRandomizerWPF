Html Help Compiler Error Message Reference

HHC3014: An information type in your project hasn't been created as part of a category.

Problem: An information type has been specified in a .HHI file that is not contained within a category. The .HHI file is a file created only the HTML Help Workshop 1.2 Preview, which was distributed at the WinWriters Help Conference in February 1998. The use of this file was dropped before HTML Help 1.2 was actually released.

Result: The information type may not work correctly.

Solution: This problem only occurs with projects created with the HTML Help Workshop 1.2 Preview, which created a .HHI file for information types. The actual 1.2 release version of HHW reads .HHI files, but does not create them (they are not officially supported). To resolve the problem, copy the information in the .HHI file into the .HHC file, and delete the reference to the .HHI file in the .HHC file. Alternatively, recreate the information types with a more current version of the HTML Help Workshop.
