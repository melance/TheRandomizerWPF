Html Help Compiler Error Message Reference

HHC3010: Sample staging directory not specified in [OPTIONS] section of the project file. Sample "filename" will not be built.

Problem: The project file does not contain the necessary information for building samples. Samples are an unsupported HTML Help feature. Microsoft has not released information on how to add sample files to a .chm file.

Note: Samples are supported in Microsoft Help 2.0, the forthcoming Microsoft help development system.

Result: Samples are not built.

Solution: Add the staging directory to the project file (bottom of the Compiler tab in the Project Properties of HTML Help Workshop).
