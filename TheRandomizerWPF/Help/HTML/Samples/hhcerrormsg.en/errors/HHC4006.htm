Html Help Compiler Error Message Reference

HHC4006: The file "filename" has already been specified in the [FILES] section of the project file.

Problem: The same file has been specified more then once in the [FILES] section.

Result: The file is only added once. This warning has no impact on the contents of the .chm file.

Solution: Remove the duplicate filename.
