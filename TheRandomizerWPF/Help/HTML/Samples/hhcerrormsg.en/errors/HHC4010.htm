Html Help Compiler Error Message Reference

HHC4010: The file "filename" in the [Text Popup] section has an unrecognized extension. Only .H and .TXT extensions may be used.

Problem: The only file extensions that are valid in the [TEXT POPUPS] section are .txt and .h. Any other extension is ignored. You must use the extension .txt for all files that contain the text popup topics, and .h for the files that contain the mapped ids from the program for which you're developing context-sensitive help.

Result: The file will not be compiled into the .chm file, and any program trying to access the file in the .chm file will fail.

Solution: Text files must have a .txt extension, and header files must have a .h extension.

Note: We could not get the compiler to generate this error. Instead, if you used the wrong extension, the compiler would silently ignore the file. You should always use .txt and .h extensions in the [TEXT POPUPS] section.
