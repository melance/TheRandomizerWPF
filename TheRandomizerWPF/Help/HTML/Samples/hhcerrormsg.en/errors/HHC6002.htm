Html Help Compiler Error Message Reference

HHC6002: The file itcc.dll has not been registered correctly.

Problem: The file itcc.dll must be registered before you can create a full-text search index, or a binary index, or support KLinks and ALinks.

Result: The .chm file cannot be created.

Solution: The problem is usually that itcc.dll is not registered correctly. To register the file from KeyTools 2.0, choose Find Files from the Tools menu, and then choose HTML Help components. After the list of components builds, select the version of itcc.dll that you want to register, and then click the Register button.

If you do not have KeyTools 2.0 or later, you will need to reinstall HTML Help Workshop.
