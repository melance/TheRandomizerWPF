Publishing the help

Precision Helper allows you to publish the help into a several widely used formats. Here is the list of these formats along with hypelinks for more informations.

List of publishing options

  • Compile the CHM format (Microsoft Compiled HTML Help)
  • Export into the WebHelp format
  • Export into an ePUB (e-book)
  • Export into the single html document
  • For informations about exporting into the Adobe PDF format click here
