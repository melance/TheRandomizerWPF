Project file selection dialog

Dialog for selecting the file of the project is one of the most important application components. In compare to the system open file dialog it provides an advanced options and the places bar, that is related to a project and to the help authoring generally.

This dialog is (starting from version 2.0) used also for selecting the topic templates.

Dialog contains list of files (intended to the selected place), a field for quick file searching (resp. filtering by the text), a file type filtering panel and the places bar.

Filtering by file type could be combined, if the SHIFT key is pressed during clicking the buttons (or pressing the SPACE key).

Dialog is always open in the place, that corresponds to the context and situation from which it is called.

Places bar items

  • Project files - the list of all files that ARE DEFINED anywhere in your help project (ie. in the table of contents, index of keywords, window type definition, etc.)
  • Unused files - the list of files from the project's folder (and its subfolders) that ARE NOT MENTIONED anywhere in your project (it can be usefull for assignment of existing files to the content items, or to the index keywords)
  • Project's folder - displays the project's folder contents (from this place you can freely browse your file system)
  • Target folder - displays the contents of the folder, where the resulting (compiled) file is located (from this place you can freely browse your file system)
  • This computer - shows the logical drives of your file system (from this place you can freely browse your file system)
  • External help - by clicking on this place, the system open file dialog is displayed and you can select a CHM file, whose content is then displayed in this place. Depending on the situation from which the "project file selection dialog" was called, you may be navigated to this place automatically by selecting a CHM file in another place of this dialog.
  • Topic templates - in this mode only the topic templates are listed here, along with the preview for currently selected template

Note:
    Almost the same dialog is displayed by opening the CHM Viewer tool, but in that case it contains also the "file preview pane".
