Properties panel

The properties panel contains tabs, that are very usefull in several major application functions.

First of all it's the Object inspector, by which you can set the parameters (values) of different elements in the project. The inspector content is amended contextually, always displayed the parameters of the active element, object, or item.
If you are editing an included file list or a context-sensitive help, the object inspector shows the file preview for the currently selected file (resp. topic).

Global search features can be found in the Search tab. Searching for entered text (words) can be performed in whole application scope or in the selected area of the project only.

In the Log file tab, you can find the last compilation protocol of currently open project or of a project that is selected in the active build list.

And last but not least, there is a Help tab on the properties panel, that is designed to display context sensitive help informations about currently focused page, control or item.

A popup menu of properties panel gives you a lot of possibilities to customize the appearance and layout of the panel and its tabs. The tabs order can be also changed by simply dragging the mouse.

Tabs on properties panel

  • Help
  • Search
  • Object Inspector
  • Log file
