Automatic insertion of topics into the table of contents

This wizard fills in the table of contents with topics from selected source (folder, list, etc.). This is usefull, when you already have some topic files and you want to add them into the contents at once.

Of course, only topics that are not already included, are inserted into the contents.

Topic sources

  • Topics from list of included files - topic files are inserted from list of included files
  • Topics from project's folder - topic files are inserted from the same folder as the project file (including subfolders)
  • Topics from selected folder - topic files are inserted from the folder you select (including subfolders)

Select the source of topic files and click on the "Next >" button to start this action.

If you want to sort the table of contents after insertion of topics, select the option "Sort table of contents if any topic added".
