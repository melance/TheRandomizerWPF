Main menu

Main menu (accessible through the big menu button in the top left corner) consists of basic application options such as creating a new project, opening an existing project, saving, options for editing, tools, etc.

Beside this, the main menu contains the list of recently used projects (see also Projects list and Welcome page) and the button for exiting the application. The application can be closed also by double-clicking the big menu button.

The main menu has no the character of context - ie. that does not change according to currently open page or navigation panel. Therefore, it does not contains all the available options and actions in, unlike the ribbon bar.


The detailed description about each of the main menu option (action) can be found in the ribbon bar topic.
