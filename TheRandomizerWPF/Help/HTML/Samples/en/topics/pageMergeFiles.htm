Merge files

In the list of merged files you define the external help files (CHM format), that you want to be also available when the user browses your help. It means that the index of keywords, the fulltext search index, and (when specified) the table of contents can be viewed by the user as it is a part of your help project.

If you want to include the table of contents of external help into your contents, you have to define an appropriate item in your table of contents.

The advantage of such definition is that the help files (including yours) are compiling separately and can be distributed separately, one without the other.

But keep in mind that if you want to use informations from the external help, it must be included in the distribution of your help files.

The list of merged files is accessible through the Settings navigation panel.

Working with merge files page

  • Adding files to the list is allowed by using the "Add" action (adds the files you select in the system open file dialog) and by the "Add near" action (it adds all the help files located in project folder and in the target folder of project compiled help file, including subfolders). You can also add files via the clipboard or the drag & drop functionality (see below).
  • Selected files can be copied to and pasted from the clipboard, and it is possible between Precision Helper application instances, as well as between other applications (Windows Explorer, text editor, etc.)
  • The same possibilities as for the clipboard are accessible through the mouse drag & drop functionality.
  • You can open the selected file directly from the list, by the View help file action, or by double clicking the file item, or by pressing the ENTER key on selected file item.
  • Quick search (filtering) possibilities are accessible through the Search field (in the top right corner)
  • Removing selected files from the list is allowed by the "Remove" action.
