Log file

In the Log file tab of the properties panel you can find the last compilation log file of the currently open project or of the project that is selected in the currently edited build list.

In addition to log file preview, there is a brief statistic and the detailed list of errors and warnings that are occured in the report log. By clicking (focusing) on the detailed list item, you are automatically focused to the appropriate line in the log file preview.

The log file popup menu options allows you to open even more detailed help about the selected error message. You can select to view selected error message help (in the Help tab) or to view whole Html Help Compiler error message reference.
