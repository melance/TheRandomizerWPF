Settings (the navigation panel)

The Settings navigation panel brings together the links to parameters, options and sections of the help project, for which there is no need of a separate comprehensive navigation panel, so as to contents or index.

By clicking a desired panel item (or by pressing the ENTER key) you open the editor page appropriate for the parameter (option, section, ...).

Items of the Settings panel

  • Project properties - opens the general project properties page
  • Window types - opens the window types editor page
  • Merge files - opens the editor of the files, that are explicitly included in the project
  • Other sections - opens the editor for direct manual editing of additional project file sections
