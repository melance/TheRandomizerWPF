Quick access toolbar

Quick access toolbar is a place for small buttons, that are always available, no matter what kind of page is opened, no matter what kind of navigation panel is active and no matter which ribbon bar tab is visible. User can customize displayed buttons via the popup menu of this toolbar. Toolbar can be placed above or under the ribbon bar, or you can completely hide it. You can find all these options in quick access toolbar popup menu.
