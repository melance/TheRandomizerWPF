Automatic insertion of keywords into the index

With help of this wizard, you can add a new keywords from topic titles into the index. Of course, only such combinations of keywords and topics, that are not already included in the index, are filled in.

You can select a topic files source, they titles will be parsed for keywords, next select if you want to divide titles into separated words (elsewhere whole title text will be inserted as one keyword) and in the end, you can choose if you want to "sort index if any keyword is added".

Topic files sources

  • Topics of current table of contents - keywords are parsed from topic files of current table of contents
  • Topics of all tables of contents in the project - keywords are parsed from topic files of all tables of contents listed in the project
  • Topics from list of included files - keywords are parsed from topic files listed in project's included files list
  • All accessible topics - keywords are parsed from topics, that are listed anywhere in the project

Select the source of topic files, possibly more options you want, and click on "Next >", to start this action.
