API Tester

API Tester is tool to quick "live" verification of context-sensitive help definition. It can test help links and jumps for context IDs, topic files and keywords.

Such a testing can be performed on resulting help of current project or on any other selected CHM (Microsoft Compiled Html Help) file. CHM file selection is the first parameter of this tool (compiled help file of current project is always set by default).

For each of jump type, there is a combo box available with current project context-help related items pre-filled.

Do not forget to compile your help project before testing with this tool.
