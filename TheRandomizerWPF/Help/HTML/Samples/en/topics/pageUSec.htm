Other sections

These editor pages can be used to manual editing of such help project file (HHP, Microsoft Html Help Project) sections, whose are not directly supported in another place of Precision Helper application. These are the INFO TYPES and SUBTYPES sections. Unfortunately, the support for these sections is not fully implemented even in the system CHM Help viewer.

Editor for each section can be opened via mouse click on appropriate item in the Settings navigation panel.

User defined sections

Precision Helper supports editing of additional user defined sections. Their names must not interfere with the standard sections of HHP project file format.

If you really want to, in the context of your projects, edit the additional sections of project help file, then append their names in the usections key of the [Global Settings] section of the application configuration file (precisionhelper.ini).

It should also be taken into account, that the html help compiler will warn you about the existence of such sections in each compilation protocol (it produces a warning message, not an error).
