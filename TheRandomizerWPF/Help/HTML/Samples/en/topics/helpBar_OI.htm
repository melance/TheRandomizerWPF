Object Inspector

With help of this object inspector, you can quickly set up and change the values of shown topic, keyword, window type, or other element (object) parameters. These parameters determine the behaviour of the object in the resulting help.
Content of the object inspector amending the context, ie. that always displays the parameters of the active element, object, or item.

For included files editor and for context-sensitive help editor, there is a file preview for selected file (resp. topic) displayed in the object inspector.

During the table of contents editing, there is a Keywords panel shown in the object inspector. It is designed to be a very useful tool for quick adding of keywords, so you can speed up the process of index definition with it.
