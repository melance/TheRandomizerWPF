Help (tab on the properties panel)

This tab of the properties panel always displays the contextual help for the active application element (list, page, control, etc.) Topics in this tab are displayed automatically, everytime the user opens another navigation panel, goes to another ribbon bar tab, activates the page with another list, and so on.

The tab has a toolbar with additional buttons, that are used to navigate Backward, Forward and to the Default help topic. By "Help table of contents" button you can open displayd help in the standard system help viewer (as a reference manual).
