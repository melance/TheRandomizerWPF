Export the help project into the Adobe PDF format

Publishing the help project into the Adobe PDF file format is realized via the export into the single html document, that can be virtually printed or converted into the PDF by any external tool.

Tutorial (variant A)

  1. First, perform the export of your help project into the single html document format.
  2. Now, the main document, you've selected in the wizard, open in your web browser (the wizard also gives you this possibility after finishing its job)
  3. Here you can select the Print option (not the quick printing, but the option that offers the printer dialog) and choose any "virtual printer", that is designed to save the printed document into the PDF file.
  4. Click the "Print" button and follow the instructions of the "virtual printer" driver. Anyway, you will be asked to enter the name and location of the resulting file in PDF format.
  5. You have just completed the export (publishing) of your project into the Adobe PDF file format.

Notes:

There are a lot of tools, that provides the "virtual printing" possibilities into the Adobe PDF file format, on the internet. For all we can mention the doPDF, PDF Factory or FinePrint.

There is also a large number of tools available for direct conversion from HTML to PDF file format.

Tutorial (variant B)

  1. First, perform the export of your help project into the single html document format (as in the variant A).
  2. Now, the main document, you've selected in the wizard, open in text editor tool, that is able to open the html file format and that can convert it into the PDF file format. For example, it is a Adobe Acrobat editor or Microsoft Office Word (version 2007 and newer).
  3. Here you select an appropriate action to Save (or to convert) the document into the PDF file format.
  4. You have just completed the export (publishing) of your project into the Adobe PDF file format.
