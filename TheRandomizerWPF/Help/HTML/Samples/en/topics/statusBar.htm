Status Bar

Status bar displays the available informations about the currently open project, about the selected items (content, index, etc.), the modification state, a hint for control under mouse, as well as informations about the ongoing operation (if any).

In addition to this, the icons of your external tools are placed on the right side of the status bar. By clicking on desired icon you open defined external application. if some content item, index item, file in the list or context-help item is selected, it is passed to executed application as a parameter.
The first icon from the left is the icon for "View resulting help" action.

In the right corner of the status bar, there is a combo box to support a zooming capability for previed topic.

If the window type page is active, the window type dimensions are displayed in the status bar.
