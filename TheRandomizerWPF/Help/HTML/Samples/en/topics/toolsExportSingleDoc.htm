Export into "the single html document" format

This wizard is designed to create a so-called single html document. Essentially it is about a merging of all your topics into one file and copying the rest of project files (images, styles, etc.) into the specified folder (with directory structure in mind). Such a document can be then converted into Adobe PDF file or opened in another text editor.

Wizard is in all aspects the same as Exporting into the WebHelp wizard, except that the templates selection combo box is filtered to single html document templates only.

In the wizard, select a template, that will be used to build resulting document, and the document location and filename. Then click "Next >" to start this action.

When the wizard successfully finishes its work, you will be asked to immediately open the newly created document in the webbrowser.

More informations about WebHelp templates can be found in Exporting into the WebHelp topic.

Detailed "step-by-step" tutorial about creating Adobe PDF document from your help project can be read in Export into Adobe PDF format topic.
