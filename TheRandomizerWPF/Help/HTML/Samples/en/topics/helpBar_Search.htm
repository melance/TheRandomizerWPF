Search

This tab of the properties panel is designed for global searching in all the project and application areas including the topic file texts.

Enter search text in the appropriate field at the top of the tab and press the ENTER key (or click the Search button). Application will find the text in all the selected areas and displays the results in the list of search results.

If you want to obtain more specific results, you can enter more words separated by spaces - the application will search only those items (topics, words, projects, etc.) containing all the words.

Selection of the search areas is realized by the drop down menu of the "Search" button.
The application searches the currently open project, the projects list and a builds list.

At the bottom of the tab is displayed an information about the number of results found.
The "Sort results" button can be used to sort the search results in ascending or descending order and it can be used to group the results by the search areas.

Viewing of the selected item (navigation) is realized by simply clicking on it (or by selecting it by arrow keys and pressing the ENTER key). Application opens the navigation panel, the page, the tab, or other place where the item is found, and there the item is focused.

Searching and the Quick launch possibilities

  • Any of the search result item can be dragged onto one of the Quick launch buttons on the welcome page.
  • By clicking the quick launch button you can then be navigated to desired item anytime later. And this is done even if the project, that the item belongs to, is not opened (in this case the actual project will be closed - of course with asking you to save your work - then the desired project will be open and the navigation to the item will be realized).
