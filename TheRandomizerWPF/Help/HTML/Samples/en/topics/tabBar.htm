Tab Bar

Tab bar is designed to speed up the switching between currently opened topics, lists and other types of pages. It has the same functionality as so-called tabbed browsing, well known from web browsers.

Tab bar can be positioned at top or at bottom of application main window. You can select style of tabs, you can synchronize first tab position with navigation bar and (of course) you can switch and / or close visible tabs (pages). You can find all options for tab bar manipulation in its popup menu.

Switching between lists can be realized (besides mouse click on the tab) by keyboard - via shortcuts Ctrl+Tab and Shift+Ctrl+Tab. Shortcut for closing current tab (page) is Ctrl+F4.

Order of tabs can be changed via drag & drop functionality.

To open a new tab, you can use the "Open a new tab" button (it is placed far left on the tab bar) or use the Ctrl+T hot-key. Behaviour of this tab bar action can be changed in Options dialog, in the Appearance - User interface section.
