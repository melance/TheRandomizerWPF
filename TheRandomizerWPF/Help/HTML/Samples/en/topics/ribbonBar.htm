Ribbon Bar

The ribbon bar is implemented in Precision Helper application, instead of classic tool bar and main menu. This modern user interface element is widely used now, because it combines (more or less) the possibilities of toolbar and menu.

Tabs on the bar are switched contextually, according to the active page (tab), but of course you can select any ribbon bar tab at any time. In ribbon bar tabs, there are placed almost all application buttons, except for a few actions, that are accessible through the popup menus of that page, pane or field only.

For functionality and possibilities of the ribbon bar buttons, you will find more detailed help in specific page and navigation bar topics rather, then here. Here you can find the brief overview of the ribbon bar tabs and their action buttons (controls).

Home

  • Home - this opens the welcome page of the application
  • New - creates a new project (submenu contains the list of available project templates)
  • Open - opens an existing project, or one of the recently used
  • Save - saves current project, or saves it to another file
  • Close - closes currently open project
  • Exit - exit from application
  • Properties - opens the project properties page
  • Contents - opens the Table of contents navigation panel
  • Index - opens the Index of keywords navigation panel
  • Contexts - opens the Context help navigation panel
  • Files - opens the Included files navigation panel
  • Settings - opens the Settings navigation panel
  • Projects list - opens the Projects list navigation panel
  • Search - shows the global Search tab
  • Inspector - shows the Object inspector tab
  • Run - saves the current project, compile it (into the CHM format) and displays the resulting help
  • Compile - compile the current project (into the CHM format)
  • View - displays the resulting help (compiled into the CHM format), if any
  • Log file - shows the last available log file for current help project
  • Builds list - opens the Builds list navigation panel
  • Extensions - a section with add-ons that are dedicated for common usage

Contents

  • Add heading - adds a new heading into the table of contents
  • Add topic - adds a new topic into the table of contents
  • Delete - removes the selected items
  • Moving the items - options for moving items within the contents structure (move up, move down, move left, move right)
  • Duplicate - duplicates selected items
  • Sort - sorts the current selected level (items in current branch without the subitems) of the table of contents
  • Auto add topics - opens the wizard for Automatic insertion of topics into the contents
  • Paste - paste the items from clipboard into the table of contents
  • Cut - cuts the selected topics from the table of contents into the clipboard
  • Copy - copies the selected topics into the clipboard
  • Select a topic - options for assigning a topic file to the current content item
  • Inspector - shows the Object inspector for current topic
  • Edit - opens the current topic in associated external editor
  • Print - prints the topic
  • Preview - print preview of current topic
  • Find - finds text on currently shown topic page
  • Text size - option for displayed text size of current topic page
  • Zoom in - zooms the topic page in
  • Zoom out - zooms the topic page out
  • Extensions - a section with add-ons that are dedicated for the table of contents

Index

  • Add key - adds a new keyword into the index
  • Delete - removes the selected items
  • Moving the items - options for moving items within the index structure (move up, move down, move left, move right)
  • Duplicate - duplicates selected items
  • Sort - sorts the current selected level (items in current branch without the subitems) of the index
  • Auto add keywords - opens the wizard for Automatic insertion of keywords into the index
  • Paste - paste the items from clipboard into the index of keywords
  • Cut - cuts the selected keywords from the index into the clipboard
  • Copy - copies the selected keywords into the clipboard
  • Select a topic - options for assigning a topic file to the current index item
  • Inspector - shows the Object inspector for current keyword
  • Edit - opens the current topic in associated external editor
  • Print - prints the topic
  • Preview - print preview of current topic
  • Find - finds text on currently shown topic page
  • Text size - option for displayed text size of current topic page
  • Zoom in - zooms the topic page in
  • Zoom out - zooms the topic page out
  • Extensions - a section with add-ons that are dedicated for an index of keywords

Build

  • Save build - saves the currently edited build list for later use
  • Add projects - adds all projects from your projects list into the current build list
  • Open - opens the project that is selected in the build list for editing in Precision Helper
  • Run - compiles the project that is selected in the build list and displays its resulting help
  • Compile - compiles the project that is selected in the build list
  • View - displays the resulting help of the selected project (in the build list), if any
  • Compile all - compiles all projects in the build list (it means all visible and checked projects)

Publishing

  • Run - saves the current project, compile it (into the CHM format) and displays the resulting help
  • Compile - compile the current project (into the CHM format)
  • View - displays the resulting help (compiled into the CHM format), if any
  • Log file - shows the last available log file for current help project
  • Builds list - opens the Builds list navigation panel
  • Export WebHelp - publish the current project into the WebHelp format (an online help)
  • Export Single doc - publish the current project into the single html document format (that is suitable for conversion into the Adobe PDF format)
  • Export ePUB - publish the current project into an ePUB (e-book) format
  • For informations about exporting into the Adobe PDF format click here
  • Extensions - a section with add-ons that are dedicated for publishing functions

Tools

  • Auto add topics - opens the wizard for Automatic insertion of topics into the contents
  • Auto add keywords - opens the wizard for Automatic insertion of keywords into the index
  • API Tester - opens the API Tester tool
  • CHM Viewer - opens the CHM Viewer
  • Import CHM - creating a new project by decompiling existing help file (in CHM format)
  • Import projects - searches for projects in selected folder and adds them into your projects list
  • Options - opens the Options dialog (application preferences)
  • Colors - quick selection of application color scheme
  • Language - quick selection of application user interface language
  • Extensions - a section with add-ons that are dedicated as a common tools

Help

  • Html help - opens the Microsoft Html Help Workshop reference
  • HHC errors - opens the Html Help Compiler error message reference
  • Product - opens the product web page
  • Web - opens the "Precision software & consulting" web page
  • Contact - opens the e-mail client window to contact a product support via e-mail
  • Help - shows the context sensitive help for the currently active element in the application
  • Contents - opens the help reference for Precision Helper product
  • About - shows the About dialog
  • Extensions - a section with add-ons that are dedicated for the help tab
