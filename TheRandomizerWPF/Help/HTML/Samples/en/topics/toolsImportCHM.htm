Import CHM

This wizard decompiles the selected CHM (Microsoft Compiled Html Help) file archive into the specified folder and creates a new help project in HHP (Microsoft Html Help Project) format.

The required input parameters are only the path to CHM file and the file name (including path) of newly generated project.

So, in the wizard dialog select th CHM file, filename of new project and click on "Next >" to start this action.


We strongly recommend to generate a new project in EMPTY folder. In the other way, you risk an incorrect results in newly created project definition (the old files in selected folder may be included in new definition).
Similar recommendation is also relevant for the location of decompiled CHM file - for a new project always use a different folder than this, where decompiled CHM file is located.

On the other hand, if you are repeatedly testing a decompilation of the same CHM file, you can leave the source (CHM) and destination (a new project) parameters of the wizard as they are.
