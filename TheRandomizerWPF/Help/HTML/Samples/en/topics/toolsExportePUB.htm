E-book publishing - export into the ePUB format

This wizard is designed to create an e-book from your help project. Precision Helper supports an export to the ePUB format, that is one of the most widely used formats for distribution of electronic books.

The process of publishing is simple. In the wizard, select a file location for the newly created e-book (i.e. C:\test\myebook.epub). Then click "Next >" to start this action.

When the wizard successfully finishes its work, you will be asked if you want to immediately open the newly created e-book in the default reader.

E-book readers

E-books can be viewed on a different devices (PCs, mobile phones, a dedicated readers) and using a different browsers. More information about this format can be obtained on the website such as Wikipedia: http://en.wikipedia.org/wiki/EPUB.
