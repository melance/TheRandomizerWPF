The Welcome page

The Welcome page of Precision Helper product gives you such a possibilities as creating a new project, opening the project from file, the list of recently opened projects and other usual options.

In addition to this, there is a very usefull section - the Quick launch. It provides the user a great flexibility of navigation to the previously found items and quick execution of actions, that are available from the main menu or the ribbon bar.

You can freely move the buttons in the "Quick launch" section via drag & drop functionality. New action can be assigned to the button by simply clicking the mouse or by using the popup menu of the button.

You can even assign any search result item to the button (via drag and drop). Clicking such a button will result to a navigation to an assigned item, even if desired project is not currently open (in this case, the current project will be closed - of course, along with asking you for saving the changes - then the desired project will be opened and the assigned item will be navigated).

The behavior and appearance of "welcome page" can be changed in the Options dialog in the "General - Project defaults" and the "Appearance - User interface" sections.
