Customization options

Precision Helper offers a plenty options to customize the user interface and the functionality to your needs. You can change the color scheme, layout of the navigator and inspector panes, you can customize the templates for publishing the help, use application add-ons, etc., etc.

Starting from version 2.0 you can use also the application extensions

Here is the brief list of the product areas you can customize:

What parts of the product are customizable?

  • User interface visual styles
  • Navigation panels layout and visibility
  • Templates for exporting into the WebHelp and other formats
  • External tools for editing and browsing the topics, editing images and the file manager
  • Topic templates
  • Project templates
  • Syntax highlighting schemes for topic editor
  • Keystrokes for inserting the code snippets into edited topic
  • Code completion items for topic editor
  • Application extensions (add-ons for application, for project, for topic editor, ...)
