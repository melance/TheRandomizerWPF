Included files (the editor)

With help of this editor and the Included files navigation panel, you can edit the list of all files explicitly included in your resulting help.

Using the navigation panel options, you can easily filter the file list, so only the files, you want to work with, are visible. Additional quick searching (filtering) by the text is available directly in this editor (in the top right corner).

There are an adding, removing, viewing, copying to clipboard and even more options you can use in this editor. They are described below.

Working with the "included files" editor

  • Add files - adds files to the list via the system open file dialog
  • Add files from folder - adds all topic files from the folder you select in system "browse for folder" dialog (including subfolders) If you have switched on the filter by file type (in the navigation panel), then the files of that type are added, whether it is the topics, images or other selected type
  • Add project files - adds files to the list via the project file selection dialog
  • Add used files - adds topic files, that are used anywhere in the project (ie. in the contents, index, window type definition, etc.)
  • Remove - removes the selected files (of course, this is not a physical file deletion)
  • Copy to clipboard - copies the selected files list to the clipboard
  • Paste from clipboard - put the files from the clipboard to the list
  • Inspector - shows the Object inspector for currently selected file (a file preview is displayed in this context)
  • Edit - opens the selected file in associated external editor. In drop down menu of the "Edit" button, there are all associated applications available (this means the predefined editor and browser and applications that are associated with the extension of currently selected file).

Editor offers an effective possibilities by using the clipboard and by using the drag & drop functionality, both between the instances of Precision Helper, and between other applications.

The list accepts the files dragged from Windows Explorer, the list of files dragged from the text editor or spreadsheet, etc. Likewise, you can drag the files from Precision Helper into a spreadsheet or text editor.

Drag & drop feature uses all the columns in the list. This implies the need to turn off the "Show relative file pathes" option in preferences dialog, when you want to transfer selected files including the entire file path (and not just the path related to the project).

Similarly, the clipboard works, but only the file name (including path) is transfered to other applications.
