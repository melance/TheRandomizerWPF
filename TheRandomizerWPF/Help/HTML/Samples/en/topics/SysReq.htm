System requirements

Precision Helper is designed for Microsoft Windows platform. It has no special hardware nor software requirements. It is distributed either as a standard setup package or in the form of "zip archive" for portable installation. More about installation can be found in the Installation help topic.


Hardware:
IBM-compatible PC
256 MB RAM
  25 MB free disk space
Operating system:
Microsoft Windows 2000/XP/Vista/7
Other requirements:
Microsoft HtmlHelp Workshop (for compiling into the CHM format)
Microsoft Internet Explorer 6.0+ (for previewing of topics)
