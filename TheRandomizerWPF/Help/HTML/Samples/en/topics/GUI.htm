The User interface

The aim of topics in this chapter is to briefly inform about usage and functionality of application elements in those areas where the user is currently located. These topics are automatically displayed in the Help tab, always when a user opens another navigation bar, sets to the other tab in the ribbonbar, activates the page with another lists, etc.

If you cannot find the necessary informations in this chapter, try to search such a topic in How to do it? chapter.
