Projects list

This is the first navigation panel, that is shown after product installation and it is designed to manage your favorite help projects.

Panel is divided into sections "Recently opened" and "All projects".

In the recently opened projects section, there is shown up as many projects as you choose in Options dialog.

In the all projects section, you can create folder structure. The projects can be easily moved between folders via mouse or keyboard. Projects can be quickly filtered via the Search field and, of course, the selected projects can be opened for editing in this application.

This list is also used in conjuction with builds list.

Options for the "Recently opened" section

  • Open - opens the selected project for editing in Product Helper application
  • Add to projects list - adds the selected project into your projects list (into the "All projects" section)
  • Remove - removes the selected project from "recently opened" list

Options for the "All projects" section

  • Open - opens the selected project for editing in Product Helper application
  • Add project - opens the system open file dialog and adds the selected project into the projects list
  • Add folder - adds a folder into the projects list
  • Rename - starts the editing of selected projects folder name
  • Remove - removes the selected project or folder from the list (this is not a file deletion)
  • Up - moves the selected item up
  • Down - moves the selected item down
  • Left - moves the selected item to the left
  • Right - moves the selected item to the right
  • Import - Projects - with help of this wizard, you can add all projects located in desired folder into this list (see Import projects)
  • Show toolbar - switches the visibility of projects list toolbar
