Changing the layout of user interface

Each user has its own way how to work with applications, and each user complies with another layout of the user interface elements.

Precision Helper gives you a lot of customization options in many areas. Here you can find a simple tutorial to change the layout of navigation pane, tab bar and the properties panel.

You can find the resulting layout in the picture at the end of this tutorial.

Example of changing the user interface layout

  1. First, place the navigation pane on right, with help of its popup menu (see picture).
  2. Next, place the properties panel on left, with help of its popup menu again (see picture).
  3. In a similar way, you can change the location of tab bar at bottom.
  4. Finally, set the location of the properties panel tabs at top.
    Information! the properties panel is now located on the left, so its drop down menu button is now located where the drop down menu button of the navigation pane was before.
    Similarly, you can set up the displaying of the the navigation pane buttons (on left, on right, at bottom).
