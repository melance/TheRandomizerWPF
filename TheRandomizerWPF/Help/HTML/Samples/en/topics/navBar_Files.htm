Included files

With help of this navigation panel and appropriate Included files page, you can edit the list of all files explicitly included in your resulting help.

Generally it is recommended to place all used project files in this list (except the HHP project file and the resulting CHM file), since the help compiler (for the CHM format) is not able to recognize (in some cases) all the files referenced from your topics.

The navigation panel provides various options for filtering the file list and the page offers actions for adding, deleting and other modifications of this list (including file preview possibility].

The detailed description of the included files page possibilities can be found here.

Working with Included files

  • Panel is divided into two sections. In the Views section, you can find the filter types, that can be applied on the file list. The selected filter type options are displayed in the second panel section.
  • The file list can be also filtered by text, and this is done via the Search field (in the top right corner of the page).

Filter types overview

  • Folders - filter items are the folders of your project directory structure (folders are displayed in the same manner as you created them on the disk)
  • Quick folders - filter items are the folders of your project files, but arranged in the single level list
  • File types - filter items are most commonly used file types when creating a help project (so, files in the list are filtered by the type - resp. its extensions)

Note:
    If you get in a situation where you can see the navigation panel, but any appropriate page (list) is not visible, simply click the mouse on any item of navigation panel (or press the ENTER key if the panel is focused) and the page with a file list will be opened.
